## SEO Strategy Cho Tools

### 1. Keyword Strategy

Mỗi tool có:
- **Primary keywords**: 5-10 keywords chính
- **Long-tail keywords**: Nhiều variations
- **LSI keywords**: Tự động generate
- **Competitor keywords**: So sánh với đối thủ

### 2. Content Strategy

- **Tool name** trong H4 heading
- **Description** với keywords tự nhiên
- **Keywords list** hiển thị để Google index
- **Backlinks** với anchor text đa dạng
- **CTA links** "Sử dụng ngay", "Xem ngay"

### 3. Structured Data

#### SoftwareApplication Schema
```json
{
  "@type": "SoftwareApplication",
  "name": "Tên Tool",
  "applicationCategory": "GameApplication",
  "offers": {
    "price": "0",
    "priceCurrency": "VND"
  }
}
```

#### ItemList Schema
- List tất cả tools
- Position và order
- URL và description

### 4. Backlink Strategy

Mỗi tool có:
- **Direct link**: Tên tool → URL tool
- **Contextual link**: Trong description
- **CTA link**: "Sử dụng ngay", "Xem ngay"
- **Anchor text đa dạng**: 
  - Tên tool
  - Keywords
  - Action words
